🔶 CALCX — Black & Orange Calculator

A sleek, fully functional calculator built with vanilla HTML, CSS & JavaScript.

✨ Features
- Full arithmetic operations (+ − × ÷)
- Percentage and sign toggle (% ±)
- Keyboard support
- Ripple button animations
- Divide-by-zero error handling
- Live expression display

🛠️ Built With
- HTML5
- CSS3 (animations, CSS variables, grid)
- Vanilla JavaScript
